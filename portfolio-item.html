Blackjack JavaScript

The technologies used to create this game were HTML, CSS, jQuery and vanilla JavaScript. The reason behind this project was because I really enjoy playing blackjack. I use to play a game online called Hit or Stand, and always thought it would be amazing to create my own Blackjack game. One of the more difficult tasks of this project was thinking of the logic behind when the dealers and player were dealt cards. I struggled on how the dealer can automatically hit when a player decided to stand, and then letting the player know if they won the game or not.
