What to eat?

What to eat is my most recent project. I came up with this idea because most nights I have a chicken caesar salad, and when I want to spice it up, I have no idea where to look. What to eat helps users find a random recipe, and then if it looks like something they might enjoy, they can save it to their profile to reference later. The technologies I used for this project were react.js for the front end, and python and flask for the backend. This was my second project with React, and the more that I use it, the more I start to love it. It has been my go to framework for making projects now. One thing I would have done differently on this project, would be to create the backend using mongoDB and express. That way, I could change my models much more easier.
